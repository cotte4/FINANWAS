# Icon Assets for Finanwas

This directory needs the following icon files for proper SEO and PWA functionality:

## Required Icons

1. **favicon.ico** (16x16, 32x32, 48x48)
   - Standard browser favicon
   - Should be placed in `/public/favicon.ico`

2. **icon.png** (192x192)
   - PWA icon for Android devices
   - Should be placed in `/public/icon.png`

3. **icon-512.png** (512x512)
   - PWA icon for Android devices (larger size)
   - Should be placed in `/public/icon-512.png`

4. **apple-icon.png** (180x180)
   - iOS home screen icon
   - Should be placed in `/public/apple-icon.png`

5. **og-image.png** (1200x630)
   - Open Graph image for social media sharing
   - Should be placed in `/public/og-image.png`

## Optional but Recommended

6. **screenshot-desktop.png** (1280x720)
   - Desktop screenshot for PWA installation
   - Should be placed in `/public/screenshot-desktop.png`

7. **screenshot-mobile.png** (750x1334)
   - Mobile screenshot for PWA installation
   - Should be placed in `/public/screenshot-mobile.png`

## Design Guidelines

- **Brand Colors**: Use Finanwas brand colors (primary: #000000)
- **Logo**: Should include the Finanwas logo or wordmark
- **Consistency**: All icons should have consistent visual style
- **Transparency**: PNG files should have transparent backgrounds where appropriate
- **Quality**: Use high-quality images that are sharp at all sizes

## Tools for Icon Generation

You can use these tools to generate icons from a single source:
- [Real Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
